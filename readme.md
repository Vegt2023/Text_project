Text Analyzer

This project is a simple text analysis application written in Python.
The program allows a registered user to log in, select one of several predefined texts, and receive statistical analysis of the chosen text.

Features

The application provides:

User authentication (username and password pair)

Selection of one text out of several available texts

Text statistics:

Total number of words

Number of titlecase words

Number of uppercase words

Number of lowercase words

Number of numeric strings

Sum of all numbers found in the text

A word-length histogram showing how many words have each length
